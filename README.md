- **Frontend:** React (v19) with TypeScript
- **Build Tool:** Vite
- **Styling:** SCSS Modules
- **API:** Axios + OpenAPI-generated types (@hey-api/openapi-ts)
- **Quality & Automation:** ESLint, Husky, lint-staged
- **Deployment:** Netlify (hosting & CI/CD)
